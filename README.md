# 📝 To-Do List Web App

A simple and responsive To-Do List web application built with React and Tailwind CSS. It allows you to easily manage your daily tasks.

## 🚀 Features

- ➕ Add new tasks
- ✏️ Edit existing tasks
- ✅ Mark tasks as complete/incomplete
- 🗑️ Delete tasks
- 🔍 Toggle to show/hide completed tasks
- 💾 Tasks persist in local storage
- 🎯 Responsive design with scrollable task list

## 🛠️ Tech Stack

- React
- Tailwind CSS
- JavaScript

## 🛠️ Installation & Setup

Follow these steps to get the project running on your local machine:

### 1. Clone the repository

```bash
git clone https://github.com/CipherHitro/To-Do-List-React.git
cd To-Do-List-React
```
### 2. Install dependencies

```bash
npm install
```
### 3. Start the development server

```bash
npm run dev
```
